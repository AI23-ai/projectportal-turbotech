2. Instant filtering for schedule of values
3. Assembly-based efficiency

1. AI automation eliminates manual clicking
2. 76% speed advantage over competitors

### Potential Market Overlap
- Both target construction estimators (McCormick: trades, Togal: general + trades)
- Both emphasize speed/efficiency vs Excel/Bluebeam
- Both have auto-count capabilities
- Both reduce manual work

### Different Use Cases?

**McCormick Sweet Spot:**
- Larger jobs requiring detailed schedule of values breakdowns
- Need to filter by floor, area, system for customer questions

- Assembly-based workflows important

**Togal Sweet Spot:**
- Speed is #1 priority (76% faster = more bids placed)
- Cloud collaboration essential (teams working remotely)
- AI-powered search across many plan sheets
- General contractors + multiple trades
- Modern web-based workflow preferred
