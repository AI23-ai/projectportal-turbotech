## Product Overview

###What is Togal.AI?

**Definition:** Togal.AI is a cloud-based AI-powered construction estimation and takeoff software that combines traditional manual takeoff tools with advanced artificial intelligence to dramatically accelerate the bidding and pre-construction process.

**Company Background:**
- Company name "Togal" means "builder" in Irish Gaelic
- Founded by Patrick Murphy (construction background)
- Built by estimators for estimators
- Mission: Automate tedious clicking and counting tasks

**Technology Platform:**
- Cloud-based (no downloads required)
- Web browser access
- No high-end hardware needed
- AI/ML-powered image recognition and pattern detection

## Competitive Analysis

### vs. Bluebeam Revu

**Bluebeam Positioning:**
- PDF management and editing tool
- Markup and collaboration focused
- NOT specifically designed for estimating

**Bluebeam Pricing:**
- Basics: $260/year per user
- Core: $330/year per user
- Complete: $440/year per user

**Togal Advantages Over Bluebeam:**
1. **AI capabilities** - Bluebeam has NONE (image search, text search, Togal.CHAT)
2. **Purpose-built for takeoff** - Bluebeam is a general PDF tool
3. **Performance** - Users report Bluebeam is "slow" with "frequent crashes"
4. **Instant identification** - Bluebeam requires manual configuration for calculations
5. **Support** - Users cite "lack of customer support" from Bluebeam
6. **Modern features** - Curve takeoffs, split/merge, floorplan AI, auto-naming (all missing in Bluebeam)
7. **Revision tracking** - Bluebeam uses only blue/red overlays (hard to distinguish), Togal has better system

**Bluebeam Advantages:**
- Best-in-class collaboration through Bluebeam Studio
- Greater robustness for general PDF work
- More markup tools
- Document hyperlinks

**Learning Curve:**
- Bluebeam: Challenging, especially for those unfamiliar with PDF software
- Togal: Intuitive platform (per user reviews)

---

### vs. PlanSwift

**PlanSwift Positioning:**
- Quantity takeoff software for construction estimating
- Point-and-click on plans to quantify and measure materials
- Custom-designed for estimates
- Desktop software

**PlanSwift Pricing:**
- $1,749/year annual subscription
- Includes support, software updates, 2 hours of training
- One-time purchase option available (with upgrade fees)

**Togal Advantages Over PlanSwift:**
1. **Cloud-based vs PC-only** - Togal accessible anywhere, PlanSwift Windows only
2. **AI automation** - PlanSwift still requires manual clicking for each item
3. **Real-time collaboration** - PlanSwift lacks this
4. **File flexibility** - Togal accepts all types including hand-drawn, PlanSwift more limited
5. **Modern UX** - Togal web-based, PlanSwift desktop legacy software
6. **Auto-naming** - Togal has it, PlanSwift doesn't
7. **Support response** - Togal 20-min guarantee vs PlanSwift standard support
8. **Curve takeoffs** - Togal includes, PlanSwift limitations
9. **Smart fill tools** - Togal advantage

**PlanSwift Advantages:**
- **Customization options** - More extensive than Togal
- **Integrated estimating tools** - PlanSwift includes pricing/estimating, not just takeoff
- **Drag-and-drop assemblies** - Similar to McCormick
- **Plugin integrations** - Extensive ecosystem
- **Lower annual cost** - $1,749 vs $2,999 (Growth plan)

**Positioning:**
- PlanSwift: Feature-rich desktop software with estimating built-in
- Togal: Modern cloud AI-first solution focused on speed via automation

## Workflow (Based on Available Information)

### Standard Process:
1. **Upload** - Import project plans (all file types supported)
2. **Auto-organize** - AI names and organizes documents into sets/folders
3. **Autoscale detection** - Software calibrates plans automatically
4. **AI-expedited takeoff** - Click Togal Button for automated counting/measuring
5. **Manual takeoff tools** - Use smarter manual tools as needed
6. **Collaborate** - Real-time teamwork with change tracking
7. **Export** - Quantified data to estimation software or Excel
